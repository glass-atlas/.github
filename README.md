# Glass Atlas · .github repository

This repository powers the Glass Atlas organisation profile on GitHub. The content at `profile/README.md` is rendered on the org profile page.

## Repo structure
- **`profile/README.md`** — Organisation profile content (public-facing). This is what visitors see on the org page.
- **`README.md`** — This file. Maintainers guide for this repo.

## How to update the profile
- Edit `profile/README.md` and open a PR.
- Keep the tone factual and developer-focused.
- After merge, the profile updates automatically on GitHub.
